阀门报价系统 - 部署清单
========================================

✅ 必须包含的核心文件：

📁 backend/ (后端服务目录)
├── main.py                    # 主服务文件
├── ocr_config.py              # OCR配置
├── requirements.txt           # Python依赖
├── convert_excel_to_csv.py    # Excel转换模块
├── valve_model_generator.py   # 阀门型号生成
├── generate_quotes.py         # 报价生成
├── default_rules.py           # 默认规则管理
├── csv_utils.py               # CSV工具
├── ocr_correction.py          # OCR修正
├── accounts.txt               # 用户账户文件
└── merchant_data/             # 用户数据目录

📁 frontend/ (前端界面目录)
├── index.html                 # 主页面
└── app.js                     # 前端逻辑

📁 tupian/ (OCR引擎目录 - 重要！)
├── tesseract.exe              # Tesseract OCR可执行文件
├── tessdata/                  # OCR训练数据目录
│   ├── chi_sim.traineddata    # 中文简体训练数据
│   ├── eng.traineddata        # 英文训练数据
│   ├── chi_tra.traineddata    # 中文繁体训练数据
│   └── [其他语言数据文件]
├── libarchive-13.dll          # OCR依赖库
├── libcairo-2.dll             # OCR依赖库
├── libfreetype-6.dll          # OCR依赖库
├── libpng16-16.dll            # OCR依赖库
├── libtesseract-5.dll         # OCR核心库
├── libstdc++-6.dll            # OCR依赖库
├── libwinpthread-1.dll        # OCR依赖库
└── [其他OCR相关DLL文件]

📄 部署脚本
├── 快速部署.bat               # Windows部署脚本
├── 快速部署.sh                # Linux/macOS部署脚本
├── 部署说明.md                # 详细部署说明
├── 启动说明.md                # 使用说明
└── 部署清单.txt               # 本文件

========================================

⚠️ 重要提醒：

1. OCR文件完整性
   - tupian/ 目录必须完整包含
   - tesseract.exe 文件必须存在
   - tessdata/ 目录必须包含训练数据
   - 所有DLL依赖文件必须存在

2. 目录结构
   - 保持原有的目录结构
   - 不要修改文件路径
   - 确保相对路径正确

3. 权限设置
   - 确保有读写权限
   - Linux/macOS需要给部署脚本执行权限

4. 环境要求
   - Python 3.8+
   - 足够的磁盘空间
   - 网络连接（安装依赖时）

========================================

🚀 快速部署步骤：

Windows:
1. 双击运行 "快速部署.bat"

Linux/macOS:
1. 给脚本执行权限: chmod +x 快速部署.sh
2. 运行脚本: ./快速部署.sh

手动部署:
1. cd backend && pip install -r requirements.txt
2. python ocr_config.py (测试OCR)
3. python main.py (启动后端)
4. cd ../frontend && python -m http.server 8000 (启动前端)

========================================

📞 技术支持：
如遇到问题，请检查：
- Python版本是否符合要求
- 所有文件是否完整
- OCR功能是否正常
- 端口是否被占用
